====================================================
🧴 SKIN DISEASE PREDICTOR
====================================================

📌 PROJECT DESCRIPTION
This is a Machine Learning based web application that
predicts skin diseases from uploaded images using a
trained TensorFlow/Keras model and Flask backend.

----------------------------------------------------

🚀 FEATURES
✔ Upload skin images
✔ Predict disease using AI model
✔ Fast Flask API response
✔ Simple and clean UI
✔ Image preprocessing included

----------------------------------------------------

🛠️ TECHNOLOGIES USED
- Python
- TensorFlow / Keras
- Flask
- HTML, CSS, JavaScript
- OpenCV / PIL
- NumPy

----------------------------------------------------

📁 PROJECT STRUCTURE
skin-disease-predictor/
│
├── model/
│     └── skin_model.keras
│
├── static/
├── templates/
│     └── index.html
│
├── app.py
├── requirements.txt
└── README.md

----------------------------------------------------

⚙️ HOW TO RUN

1. Install dependencies:
   pip install -r requirements.txt

2. Run Flask app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

----------------------------------------------------

🧠 HOW IT WORKS
- User uploads skin image
- Image is preprocessed
- CNN model analyzes image
- Disease is predicted
- Result is displayed

----------------------------------------------------

📦 MODEL FILE
skin_model.keras (must be present in model folder)
